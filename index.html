Hieronder zie je opdrachten die we gemaakt hebben en ook waarmee we gewerkt hebben. Van leren werken met een Raspberry Pi tot het programmeren in Python en een netwerk in elkaar knutselen!

Dit is de opbouw van de raspberry pi, hier hebben we verschillende oefeningen op gemaakt. De raspberry pi is een diverse single board computer gebassert op een ARM-processor. We kunnen hier ook vanalles op aansluiten van een LCD tot een Stappenmotor!

Dit is een realisatie van een website, we hebben hierbij de website opgebouwt met verschillende koppen. Al deze koppen hebben we via grid in css een plek gegeven en zo kwamen we tot dit resultaat.

Het eerste semester hebben we geprogrammeerd in Python!

Dit is het project van onze projectweek hierbij hebben we een netwerk moeten opstarten met een aantal servers we werden in groepen verdeeld en we moesten op een gameserver tegen elkaar spelen. Ook moesten we een apache server voor een team website en een FTP server voor bestanden per speler te kunnen opzetten..
